# Gerenciador de Tarefas

Aplicativo mobile de gerenciamento de tarefas desenvolvido com React Native e Expo.

# Integrante

- Leonardo Davi Pereira Torres

# Funcionalidades

- Login com usuário e senha
- Listagem de tarefas
- Adicionar nova tarefa
- Ver detalhes e editar uma tarefa
- Alterar status da tarefa (pendente ou concluída)
- Excluir tarefa
- Tela de configurações

# Tecnologias utilizadas

- React Native
- Expo SDK 54
- React Navigation

# Como instalar e rodar o projeto

# Requisitos

- Node.js instalado
- Expo Go instalado no celular

# Passos

1. Clone o repositório:
git clone https://github.com/leonardodptorres020-dotcom/gerenciador-tarefas.git

2. Entre na pasta do projeto:
cd gerenciador-tarefas

3.  Instale as dependências:
npm install --legacy-peer-deps

4. Rode o projeto:
npx expo start

5.  Escaneie o QR code com o Expo Go ou abra no navegador apertando *W*
